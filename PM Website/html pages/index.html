Consultant Information

Need a consultant? Not sure what type of consultant you need?

  • Database Consultant
    • System Design Help:Design help with user interface layout, logos, graphics, images, etc.
    • PPT Design Help:
      Design help with PPT template, layout, graphics, images, etc.
    • System Design Feedback:
      Feedback on system design, usability, and user experience.
    • PPT Design Feedback:
      Feedback on design, look, feel, and flow of PPT presentation.
  • Design Consultant
    • Design Help:Help with ERD or the logical design.
    • SQL Help:Help writing SQL statements.
    • Validation Help:Review of ERD, Logical Design, or SQL statements.
    • Testing Help:Help testing SQL statements and database system.
  • Programming Consultant
    • Programming Logic Help: Help with the logic design, data flows, and pseudocode.
    • Coding Help: Help with writing code according to the logical design.
    • Programming Syntax Help: Help with syntax and programming structures.
    • Testing and Evaluation: Test programs and evaluate output of system.
  • Business Writing Consultant
    • Proofread PPTs:Proofreading and editing PPT slides for spelling and grammar.
    • Proofread Final Report:Proofreading and editing final report for spelling and grammar.
    • PPT Content Feedback:Read and provide constructive feedback on PPT slides.
    • Report Content Feedback:Read and provide constructive feedback on final report.
